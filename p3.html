Voltar Document

Uma regra css pode ser definida com a seguinte sintaxe


        Seletor{
            Propriedade1: Valor;
            Propriedade2: Valor;

            PropriedadeN: Valor;
        }

Podemos incluir código css de 3 formas diferentes

CSS externo
Quando definidos as regras css a serem utilizadas por meio de um arquivo externo .css, vinculando ele ao html por meio da tag < link > no head do documento html
CSS Incorporado "no documento"
Quando definimos as regras css por meio da tag <style >
CSS inline
Quando definimos código css de estilizacao no proprio elemento HTML, como: Texto de cor violeta com css inline
